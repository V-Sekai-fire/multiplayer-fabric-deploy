## Key Design Principles

### Authority Invariant
- Only the zone whose Hilbert code contains pos(x,y,z) executes CMD_INSTANCE_ASSET
- Non-authority zones forward the packet
- Prevents duplicate instancing across zones

### RISC-V Sandbox Boundary
- Scripts in uploaded scenes execute INSIDE the VM, not in zone process
- Eliminates need for script whitelist/pre-processing
- Replaces VSK approach with hardware-level isolation

### Interest Ghost Pattern
- Authority zone broadcasts CH_INTEREST to AOI_CELLS
- Neighbouring zones receive ghost (replica), no re-fetch
- One RTT latency for interest propagation

### Certificate Pinning
- WebTransport uses shared cert hashes (picoquic native)
- No OAuth; strictly certificate-based mTLS
- Works across macOS, Windows, Linux without browser
